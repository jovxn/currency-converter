# Currency Converter

A simple Chrome extension that lets you highlight any currency text on a webpage and instantly convert it.

## Features

- Highlight text like "$50" or "100 euros" and get instant conversions
- Supports 24 currencies using exchangerate-api
- Light and dark theme
- Keyboard shortcut: `Alt+Shift+C`

## Setup

1. Get a free API key from [ExchangeRate-API](https://www.exchangerate-api.com/)
2. Load the extension in Chrome:
   - Go to `chrome://extensions`
   - Enable **Developer mode**
   - Click **Load unpacked** and select this folder
3. Click the extension icon and enter your API key
